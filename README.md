# ansible-mongodb
Ansible Project to install and configure MongoDB & Users in Database

# First download Repo
$ git clone git@github.com:TecnoRV/ansible-mongodb.git

# Run the playbook in your working path
----------------
$ ansible-playbook mongodb.yml

PLAY [mongoservers] **************************************************************************************************************************************************************************************************************
